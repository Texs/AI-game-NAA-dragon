# Contributing to AI-game-NAA-dragon

Thank you for your interest in contributing to the AI-game-NAA-dragon project! We appreciate your time and effort in making this project better.

## Getting Started

1. Fork the repository
2. Clone your forked repository
3. Create a new branch for your feature or bug fix
4. Make your changes
5. Test your changes thoroughly
6. Commit your changes with a clear commit message
7. Push your changes to your forked repository
8. Submit a pull request

## Development Setup

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)
- Python (v3.6 or higher)

### Installation

1. Install Node.js dependencies:
   ```bash
   npm install
   ```

2. No additional Python setup needed (uses standard library only)

## Running Tests

To run JavaScript tests:
```bash
npm test
```

To run Python tests:
```bash
python test.py
```

## Code Style Guidelines

### JavaScript
- Use 4-space indentation
- Use single quotes for strings
- Use semicolons at the end of statements
- Write JSDoc comments for all functions
- Follow the ESLint configuration provided in `.eslintrc.json`

### Python
- Follow PEP 8 style guide
- Use docstrings for all functions
- Use descriptive variable names
- Keep functions focused and small

## Project Structure

- `app.js` - Main JavaScript application logic
- `app.py` - Main Python application logic
- `test.js` - JavaScript unit tests
- `test.py` - Python unit tests
- `index.html` - Main HTML page for the CI/CD project
- `package.json` - Node.js package configuration
- `.eslintrc.json` - ESLint configuration

## Pull Request Process

1. Ensure your code follows the style guidelines
2. Update documentation if necessary
3. Add tests for new functionality
4. Make sure all tests pass before submitting
5. Provide a clear description of your changes in the pull request

## Reporting Issues

When reporting issues, please include:
- A clear title and description
- Steps to reproduce the issue
- Expected vs. actual behavior
- Any relevant error messages
- Environment information (OS, Node.js version, etc.)

## Contact

For questions about contributing, feel free to open an issue or contact the maintainers.

Thank you again for contributing!
